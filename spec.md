# Specification

## Summary
**Goal:** Rename the brand from "Silk Splendor" to "KBR Silks" across the application.

**Planned changes:**
- Update the app header to display "KBR Silks" instead of "Silk Splendor"
- Update the HTML page title (browser tab) to "KBR Silks"
- Remove any remaining visible references to "Silk Splendor" in the UI

**User-visible outcome:** The app header and browser tab title will show "KBR Silks" with no remaining traces of the old brand name "Silk Splendor".
